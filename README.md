# AC Coder

## Sobre o Projeto

O **AC Coder** é um website moderno desenvolvido com **React + Vite**, criado para fins de freelancing e apresentação de serviços de desenvolvimento web.

O projeto tem como objetivo demonstrar competências em criação de interfaces interativas, responsivas e focadas na experiência do utilizador, além de servir como portfólio profissional para captação de clientes.

## Objetivo

* Freelancing e captação de clientes
* Demonstração de habilidades em desenvolvimento web
* Criação de presença digital profissional

## Funcionalidades

* Website interativo e moderno
* Sistema de orçamento para clientes
* Contacto direto via WhatsApp
* Design responsivo (mobile, tablet e desktop)
* Interface rápida e otimizada com Vite

## Tecnologias Utilizadas

* React
* Vite
* HTML5
* CSS3
* JavaScript
